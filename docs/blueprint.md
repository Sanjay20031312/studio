# **App Name**: BlockPay

## Core Features:

- Biometric Authentication: Secure biometric authentication for login and payments, offering a seamless user experience.
- Wallet Dashboard: Display wallet balance in crypto and loyalty points with transaction history.
- QR Code Payments: Scan QR codes for quick payments and wallet address sharing.
- Payment Method Selector: Display payment methods and allow users to select preferred method during transactions.
- Real Time Status Updates: Processing page that shows a live progress and the history of recent transactions.
- Transaction summaries: Provide a summary of transactions for users. Provide transaction success, failure, and receipt summaries.
- Intelligent amount prompts: Suggest typical amounts to pay, based on purchase history of similar items at Walmart using a generative AI tool.

## Style Guidelines:

- Primary color: Medium-dark cyan (#468B97) evoking a sense of trust, stability, and innovation for financial transactions.
- Background color: Light cyan (#E1E8EA), a desaturated tone providing a clean, unobtrusive backdrop.
- Accent color: Muted purple (#9D7FB7) used in small doses to signal secure transactions, confirmations, or status changes.
- Body font: 'PT Sans' sans-serif offering modern, clean legibility. Header font: 'Space Grotesk', a slightly futuristic sans-serif, bringing a modern feel appropriate for a forward-thinking payment system.
- Use minimalist icons representing transaction types, wallet functions, and security features. Consistent style and weight for a unified look.
- Mobile-first, tab-based navigation. Prioritize essential actions within thumb's reach. Use clear visual hierarchy for information display.
- Subtle animations and transitions to provide user feedback and maintain engagement during transaction processes.
